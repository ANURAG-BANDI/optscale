## Overview

OptScale is available either as an open-source code or as a [SaaS solution](https://my.optscale.com) hosted by Hystax team. 

Major capabilities:
- FinOps advisor and checklist
- ML leaderboards, experiment tracking, bulk run of ML/AI experiments
- ML model profiling, any PaaS or SaaS instrumentation
- Dozens of cost and performance optimization scenarios (including VM rightsizing), S3 duplicate finder and security recommendations
- Cost allocation, budget constraints, daily limits and quotas with instant alerts
- Cost anomaly detection, RI/SP & Spot Instance recommendations
- GEO cost map and network traffic visualizer
- AWS, Microsoft Azure, Alibaba Cloud, GCP and Kubernetes are supported
  
You can check OptScale [live demo](https://my.optscale.com/live-demo) to explore product features on a pre-generated demo organization.

Learn more about the Hystax OptScale platform and its capabilities at [our website](https://hystax.com).
